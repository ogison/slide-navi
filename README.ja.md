# Slide Navi

Slide Navi は、スライド資料のリハーサルを支援する Next.js アプリケーションです。PDF をアップロードして台本を作成し、スライド再生と同期したタイプライター効果や音声合成を使って練習できます。

## サマリー

- PDF をアップロードして、スライドのプレビューやページ移動を行えます。
- スライドごとに自動的にグループ化されるナレーション原稿を作成・管理できます。
- 自動再生や再生間隔の調整、タイプライター効果または音声合成を使ってリハーサルできます。
- オーディオ設定はローカルストレージに保存され、次回のリハーサルでも引き継がれます。

## セットアップ

1. 依存関係をインストール: `npm install`
2. 開発サーバーを起動: `npm run dev`
3. 本番ビルドを作成: `npm run build`
4. 本番ビルドをローカルでプレビュー: `npm run start`

## プロジェクト構成

- `src/app` – App Router のエントリーポイントとグローバルスタイル。
- `src/components` – スライドビューア、コントロールパネル、ヘッダーなどの UI コンポーネント。
- `src/hooks` – PDF 取り込み、スライド再生、オーディオ制御、音声合成のカスタムフック。
- `src/utils` – オーディオ生成などのユーティリティ関数。
- ルートの設定ファイル（`tsconfig.json`、`eslint.config.mjs`、`postcss.config.mjs` など）がツールチェーンを構成します。

## 主要な技術スタック

- App Router を利用した Next.js 15 と React 19。
- Tailwind CSS 4 によるユーティリティファーストなスタイリング。
- ESLint、Prettier、Husky、lint-staged によるコード品質の維持。
