# Slide Navi

Slide Navi はスライドプレゼンのリハーサルを支援する Next.js 製アプリケーションです。PDF 資料を読み込み、台本を紐づけて、スライド同期やタイプライタ風効果音・音声合成を使った練習ができます。

## 概要

- PDF をアップロードしてスライドごとのプレビューとナビゲーション操作が可能。
- スライド単位の台本管理と自動グルーピングでナレーションを整理。
- オートプレイ、タイミング調整、タイプライタ音・音声合成の切り替えで本番を想定したリハーサルが可能。
- 音声設定などのプリファレンスをローカルストレージに保存し、次回以降も同じ条件で練習可能。

## セットアップ

1. 依存関係のインストール: `npm install`
2. 開発サーバーの起動: `npm run dev`
3. 本番ビルドの作成: `npm run build`
4. 本番ビルドのローカルプレビュー: `npm run start`

## 使い方

1. **スライドをアップロード** — 「PDF」タブから資料を読み込むと、ページごとのプレビューで順番を確認できます。
2. **台本を読み込む / 作成する** — 「台本」タブで JSON 台本を貼り付けるか編集します。フォーマットの詳細は `docs/script-input-guide.md` を参照してください。
3. **リハーサルする** — ビューワーに戻り、手動操作またはオートプレイでスライドを進行。タイプライタ効果音と音声合成を切り替えたり、再生タイミングを調整できます。
4. **すぐに改善する** — 台本 JSON はその場で編集でき、パースエラーはエディター上に表示されるため修正が簡単です。

## ドキュメント

- `docs/script-json-prompt.md` — LLM に台本生成を依頼するためのプロンプトテンプレート。
